# Ralix TodoMVC

[TodoMVC](https://todomvc.com) implementation with Ralix, bundled with Parcel.

See 🌐 [live version](https://ralix-todomvc.vercel.com).

## Install

🔧 Install the app:

```
> yarn install
```

## Run

🆙 Run the local server:

```
> yarn dev
```

Open your browser at http://localhost:1234.

## Build

📦 Build the app:

```
> yarn build
```
